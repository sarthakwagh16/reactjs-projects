# ReactJS Mini Projects

This repository showcases various mini projects built using ReactJS, highlighting different aspects of React development such as state management, component composition, and API interaction.

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/sarthakwagh16/reactjs-projects.git
   ```
   
2. Navigate to a project folder:
   ```bash
   cd project-folder-name
   ```
   
3. Install dependencies:
   ```bash
   npm install
   ```
   
4. Start the project locally:
   ```bash
   npm start
   ```
